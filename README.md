# 🎲 Dice Game

A simple and interactive two-player Dice Game built using **HTML**, **CSS**, and **JavaScript**. Each player rolls a virtual dice once, and the player with the higher number wins the game.

---

## 🚀 Features

- 🎲 Two-player dice rolling
- 🎯 Random number generation (1–6)
- 🏆 Automatically declares the winner
- 🤝 Detects draw situations
- 🔄 Reset button to start a new game
- 🚫 Prevents players from rolling multiple times in a round
- 💻 Responsive and clean user interface

---

## 🛠️ Technologies Used

- CSS3
- JavaScript (DOM Manipulation)

## 🎮 How to Play

1. Click **Roll** for **Player 1**.
2. Player 2's Roll button becomes active.
3. Click **Roll** for **Player 2**.
4. The game compares both dice values.
5. The winner or draw result is displayed.
6. Click **Reset** to start a new round.
